# Delete Comment Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a × Delete button to each saved comment bubble that removes the comment from data, unwraps the highlight mark in the article, removes the bubble from the DOM, and calls markDirty().

**Architecture:** All changes live in `src/engine.js` (DOM logic inside the IIFE) and `src/engine.css` (hover-visible button style). The delete handler is wired inside `renderCommentBubble()`. No new files. After implementation, `node build.js` regenerates `dist/collab-template.html`.

**Tech Stack:** Vanilla JS (ES2020), CSS3. No libraries. Node.js `node:test` suite (utils only) is unaffected; no new tests needed.

### Task 2: JS — `deleteComment()` + wire into `renderCommentBubble()`

**Files:**
- Modify: `src/engine.js`

- [ ] **Step 3: Add `deleteComment(id)` function**

Insert immediately after `renderCommentBubble()` and before `function escapeHtml`:

```js
function deleteComment(id) {
  // 1. Remove from collab-data
  const data = getCollabData(document);
  data.comments = data.comments.filter(c => c.id !== id);
  setCollabData(document, data);

  // 2. Unwrap <mark> in #collab-content — preserve inner text, remove wrapper
  const mark = $('collab-content')
    .querySelector(`.collab-highlight[data-comment-id="${id}"]`);
  if (mark) {
    const parent = mark.parentNode;
    while (mark.firstChild) {
      parent.insertBefore(mark.firstChild, mark);
    }
    parent.removeChild(mark);
  }

  // 3. Remove bubble from sidebar DOM
  const bubble = $('collab-comments-list')
    .querySelector(`.collab-comment-bubble[data-comment-id="${id}"]`);
  if (bubble) bubble.remove();

  // 4. Mark document dirty
  markDirty();
}
```

- [ ] **Step 4: Replace `renderCommentBubble()` to wire the delete button**

Replace the entire `renderCommentBubble` function with:

```js
function renderCommentBubble(comment) {
  const list = $('collab-comments-list');
  const bubble = document.createElement('div');
  bubble.className = 'collab-comment-bubble';
  bubble.dataset.commentId = comment.id;
  bubble.innerHTML = `
    <button class="collab-comment-delete" aria-label="Delete comment" title="Delete comment">×</button>
    <div class="collab-comment-quote">${escapeHtml(comment.quote)}</div>
    <div class="collab-comment-body">${escapeHtml(comment.text)}</div>
    ${comment.images.map(img =>
      img.type === 'base64'
        ? `<img src="${img.data}" alt="screenshot">`
        : `<img src="${img.url}" alt="screenshot">`
    ).join('')}
    <div class="collab-comment-meta">${new Date(comment.timestamp).toLocaleString()}</div>
  `;
  // Wire delete button after innerHTML (avoids inline onclick in HTML string)
  bubble.querySelector('.collab-comment-delete')
    .addEventListener('click', (e) => {
      e.stopPropagation();
      deleteComment(comment.id);
    });
  list.appendChild(bubble);
}
```
